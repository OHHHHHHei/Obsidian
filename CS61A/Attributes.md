# Method Calls

通过点表达式可以调用类中的方法

# Attribute Lookup

使用点表达式的时候，python都会去寻找你访问的特性，会优先查看实例属性中是否有你访问的属性，如果没找到，就会在类属性中寻找。

# Class attribute

类内有两种属性，一个是类属性，一个是实例属性。实例属性可以在属于这个类的多个对象之间不同，而类属性则在属于这个类的所有对象中都相同，例如所有的人都是人类，但是每个人的名字都各不相同。

# Bound Methods

**定义在类中的函数，当它被一个实例所调用时，它的行为就是方法（Method），因为它会自动将该实例作为第一个参数（通常叫 `self`）传入。** 
Object + Function = Bound Method

# Attribute Assignment

点表达式可以进行属性赋值，可以给一个实例新建一个新的实例属性，即使类中已经有了同名的属性。
