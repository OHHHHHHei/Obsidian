### Python 的执行方式
- Python 既可以进行交互式对话，也可以解释执行整个 `.py` 文件。
- 加入``-i`` 可以以交互对话打开``.py``文件
---
### 多重赋值的运算顺序
- 例如 `b, a = a + b, b`，先计算右侧表达式的值，然后依次赋值给左侧的变量。
- --
### Python 的变量查找顺序
- Python 按照 LEGB 规则查找变量：Local -> Enclosing -> Global -> Built-in。
---
### 文档字符串
- Python 社区约定俗成地推荐使用三引号来定义文档字符串
- 不仅充当注释，还可以给出样例。
- `python3 -m doctest lab00.py`用来执行文档字符串测试
