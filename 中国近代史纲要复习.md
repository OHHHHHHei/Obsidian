# 为什么说鸦片战争是中国近代史的开端
1.改变了中国的社会性质，从封建社会转变到半殖民地半封建社会
2.使得中国的社会主要矛盾从地主阶级与农民阶级的矛盾变成了地主阶级与农民阶级与帝国主义与中华人民的矛盾
3.革命任务发生变化，在原本的推翻封建制度的基础上增加了反抗外国殖民者的入侵，革命性质也从农民战争变成了旧民族主义革命
4.改变了中国的发展方向，中国原本正在缓慢向资本主义社会发展，但鸦片战争使得国内资本主义无法正常发展，进而选择了社会主义。

# 洋务运动为何失败
1.洋务运动具有封建性，本质是地主阶级进行的救亡图存运动，本质还是为了维护清政府的统治
2.洋务运动对外国具有依赖性
3.洋务运动管理的腐败性，虽然是资产阶级革命，但是管理是封建式的
教训：地主阶级不能承担中国近代化的历史责任
# 戊戌维新
1.是一次爱国维新运动，贯穿爱国主义精神，推动了中华民族的觉醒
2.资产阶级性质的运动
3.是一场思想启蒙运动
## 失败原因
1.慈禧太后强大的封建主义阻挠
2.政治上不敢完全否定封建主义
3.经济上没触及封建主义土地私有制度
4.对帝国主义抱有幻想，希望依靠帝国主义殖民者帮助维新
5.没有获得人民群众的支持
# 辛亥革命失败原因
1.根本原因：在帝国主义时代，在半殖民地半封建社会的中国，资本主义建国路线是走不通的
2.没有彻底的反帝反封建的革命纲领
3.无法充分地发挥人民群众的力量
4.不能建立坚强的革命政党作为团结一切力量的核心
# 五四运动
1.是中国新民主主义革命的开端
2.在中国传播了马克思主义以及加强了马克思主义与工人阶级运动的结合
3.表现出了彻底的反帝反封建性质
4.是一次真正的人民群众运动，工人阶级开始以独立的姿态登上历史舞台
5.极大地解放了思想，传播了民主与科学的新思想
# 中国共产党
第一，建立广泛的统一战线。
第二，坚持革命的武装斗争。
第三，加强共产党自身的建设。
毛泽东指出：“==统一战线，武装斗争，党的建设==是中国共产党在中国革命中战胜敌人的三个法宝，三个主要的法宝。”
# 抗日战争
以爱国主义为核心的民族精神是中华人民抗日战争胜利的决定因素
中国共产党的作用
全民族抗战
全世界反法西斯人民的支持
## 中国共产党的中流砥柱作用
1.中国共产党积极倡导、促成、维护抗日统一战线，最大限度动员全国军民共同抗战成为凝聚全民族抗战力量的杰出组织者和鼓 舞着者
2.制定、 实施了一套完整的抗战策略和方针，提出了持久抗战的战略思想，对抗战胜利发挥了重要作用。
3.中国共产党通过游击战开辟 敌后战场，建立抗日根据地，牵制和消灭了日军大量有生力量，减轻了正面战场的压力，也为抗日战争的战略返攻准备了条件。
4.中国共产党人以自己最富于献身的爱国主义、不怕流血牺牲的模范行动，支撑起全民族救亡图存的希望，成为夺取抗战胜利的民 族先锋。
# 国民党在抗日战争
1.国民党领导的正面战场，对抗日战争的胜利做出了重要贡献。
2.国民党政府执行的是片面抗战路线，即不敢放手发动和武装民众，实行单纯的政 府和正规军的抗战；在战略战术上，没有采取积极防御的方针，而是进行单纯的阵地防御战
3.，国民党的正面战场在抗战个阶段中表现不同，其地位和作用也不同。抗战初期的 战略防御阶段，国民党政府积极抗战，正面战场在整个抗战中起了重要作用。抗战进入战略 相持阶段，其实行片面抗战，制造反共摩擦，在抗战中的地位、作用明显下降。在战略反攻 阶段，其虽坚持抗战，但对夺取抗战最后胜利的作用十分有限。
