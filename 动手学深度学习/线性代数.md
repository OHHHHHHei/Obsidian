# 张量

张量是矩阵的推广。张量是一个描述具有任意数量之后的n维数组的通用方法。向量是一个一阶张量，矩阵是一个二阶张量。

# 张量算法的基本性质

按元素的一元运算都不会改变操作数的形状。同样， 对给定具有相同形状的两个张量，任何按元素二元运算的结果都将是相同形状的张量。

# 非降维求和

有时候，调用函数计算总和或者均值的时候保持轴数不变将会变得非常有用。

```python
sum_A = A.sum(axis=1, keepdims=True)
```

在利用广播机制的时候，这个操作将会变得非常有用。

我们可以沿某个轴计算元素的累积总和
```python
A.cumsum(axis = 0)
```

这不会降低张量的维度
# 点积

对给定向量，是相同位置的按元素乘积的和

同样，我们可以先按照元素乘法，在进行求和来便是两个向量的点积

点积可以表示加权平均，一个向量为权重向量，一个为数值向量，两个向量进行点积就是加权平均。

# 范数

非正式的说，向量的范数是表示一个向量有多大。这不涉及维度，而是分量的大小。

向量范数是将向量映射到标量的函数$f$ ,给定任意向量X， 向量范数应该要满足三个性质

性质1. $f(\alpha \mathbf{x}) = |\alpha| f(\mathbf{x}).$ 伸缩性质
性质2.$f(\mathbf{x} + \mathbf{y}) \leq f(\mathbf{x}) + f(\mathbf{y}).$ 这也是三角不等式
性质3.$f(\mathbf{x}) \geq 0.$ 范数必须非负

$L_2$ 范数是
$$\|\mathbf{x}\|_2 = \sqrt{\sum_{i=1}^{n} x_i^2},$$ 
我们可以按照如下方式计算向量的范数
```python 
u = torch.tensor([3.0, -4.0])
torch.norm(u)
```
我们同样会遇到$L_1$ 范数，这表示为向量元素的绝对值之和
$$\|\mathbf{x}\|_1 = \sum_{i=1}^{n} |x_i|.$$ 
我们将绝对值函数和按元素求和组合起来可以求得
```python
torch.abs(u).sum()
```


对更一般的$L_p$ 范数
$$\|\mathbf{x}\|_p = \left( \sum_{i=1}^{n} |x_i|^p \right)^{1/p}.$$ 类似$L_2$ 范数，我们有$F$ 范数，这是矩阵元素平方和的平方根
$$\|\mathbf{X}\|_F = \sqrt{\sum_{i=1}^{m} \sum_{j=1}^{n} x_{ij}^2}.$$ 我们可以这样求得矩阵的$F$ 范数
```python
torch.norm(torch.ones((4, 9)))
```
同样也可以使用
```python
norm = torch.linalg.norm(x)
print(norm)
```
