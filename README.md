## 🛠️ 工具与工作流 (Tools & Workflow)

本知识库基于 **Obsidian** 构建，采用了以下工作流：

- **双向链接 (Bi-directional Linking)**: 关联相关概念 (e.g., `[[傅里叶变换]]` <-> `[[信号处理]]`)。
- **标签系统 (Tags)**:
    - `#todo`: 待完善的笔记
    - `#concept`: 核心概念定义
    - `#code`: 代码片段与实现
- **核心插件**:
    - **Dataview**: 动态汇总复习进度。
    - **Excalidraw**: 绘制电路图、神经网络架构图。
    - **Latex Suite**: 快速输入数学公式 (e.g., $e^{i\pi} + 1 = 0$).
