
给定2个正整数 $a$ 和 $b$ （假设 $a > b$ ）
计算 $a \div b$ 的余数 $r$
如果 $r = 0$，则b就是这两个数的最大公约数。
如果 $r \neq 0$，则令 $a = b, b = r$，并重复步骤 $1$，直到余数为 $0$。
我们可以利用切割的思想理解：
### 故事：国王的完美主义长方形庭院

想象一下，你是一位古希腊的国王，你有一个巨大的长方形庭院，长 252 米，宽 105 米。

你是一个完美主义者，想用**同样大小的正方形**地砖把整个庭院铺满，而且要求：

1. 地砖必须是**正方形**的。
    
2. 地砖要**尽可能大**（这样才气派，而且用的地砖总数最少）。
    
3. 铺满庭院后，地砖**不能切割**，必须是完整的。
    

这个“最大尺寸的正方形地砖”的边长，其实就是 252 和 105 的**最大公约数 (Greatest Common Divisor, GCD)**。

那么，怎么找到这个边长呢？

---

### 第一步：先用短边当标准试一试

你的庭院是 $252 \times 105$ 的。最直观的想法是，既然要用正方形铺，那正方形的边长肯定不能超过最短的那条边，也就是 $105$ 米。

好，我们先尝试用 $105 \times 105$ 米的地砖来铺。

你沿着长边铺设：

- 铺下第一块 $105 \times 105$ 的地砖，长边还剩下 $252 − 105 = 147$ 米。
    
- 再铺下第二块 $105 \times 105$ 的地砖，长边还剩下 $147 − 105 = 42$ 米。
    

现在，你铺了两块大地砖，庭院还剩下一个小一些的长方形区域，尺寸是 **$105 \times 42$ 米**。

**核心思想来了：** 如果存在一块能完美铺满整个大庭院的“终极正方形地砖”，那么它也必须能完美铺满你刚刚铺下的那两块 $105 \times 105$ 的区域，**并且**也必须能完美铺满剩下的那块 $105 \times 42$ 的小庭院。

反过来说，只要我们能找到**能铺满这块 $105 \times 42$ 小庭院的最大正方形地砖**，它就一定是我们要找的答案！

所以，问题被简化了：从求 `gcd(252, 105)` 变成了求 `gcd(105, 42)`。

---

### 第二步：在剩下的小庭院里重复同样的操作

现在你的问题是在 $105 \times 42$ 的庭院里找最大的正方形地砖。

我们重复刚才的步骤：用短边 42 米当标准。

- 铺下第一块 $42 \times 42$ 的地砖，长边还剩下 $105 − 42 = 63$ 米。
    
- 再铺下第二块 $42 \times 42$ 的地砖，长边还剩下 $63 − 42 = 21$ 米。
    

现在，你又剩下了一个更小的长方形区域，尺寸是 **$42 \times 21$ 米**。

问题再次被简化：从求 `gcd(105, 42)` 变成了求 `gcd(42, 21)`。

---

### 第三步：最后一步

我们继续在 $42 \times 21$ 的庭院里重复。用短边 21 米当标准。

- 铺下第一块 $21 \times 21$ 的地砖，长边还剩下 $42 − 21 = 21$ 米。
    
- 再铺下第二块 $21 \times 21$ 的地砖，长边还剩下 $21 − 21 = 0$ 米。
    

**完美！** 这次没有任何剩余了！$42 \times 21$ 的庭院被 $21 \times 21$ 的正方形地砖正好铺满。

这意味着，我们找到了！能铺满 $42 \times 21$ 区域的最大正方形地砖，边长就是 **21** 米。

既然它能铺满 $42 \times 21$ 的区域，根据我们之前的推理，它也就能铺满 $105 \times 42$ 的区域，最终也就能铺满整个 $252 \times 105$ 的大庭院。

所以，国王要找的**最大地砖边长就是 21 米**。

利用递归可以很轻易的写出gcd算法。

```C
int gcd(int a, int b) {
    if (b == 0)
        return a;
    else
        return gcd(b, a % b);
}
```
