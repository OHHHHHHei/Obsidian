#  Static and Non-static

### . 静态 (Static) 方法

- **如何定义：** 使用 `static` 关键字，如 `public static void makeNoise()`。

- **它属于谁：** 它属于 `Dog` 这个 **类**。它不属于任何一个 具体的狗对象（比如 `maya`）。

- **如何调用：** **直接使用类名** 来调用

- **示例：** `Dog.makeNoise();`

- **核心限制 (最重要)：** 静态方法 **不能** 访问非静态的实例变量 (instance variables)。

### 2. 非静态 (Non-static / 实例) 方法

- **如何定义：** **不**使用 `static` 关键字，如 `public void makeNoise()

- **它属于谁：** 它属于一个**具体创建出来的对象 (实例)**。

- **如何调用：** **必须通过一个实例（对象）** 来调用。

- **核心特性：** 实例方法 **可以** 自由访问**它自己**的实例变量。

### `non-static` (实例) 变量

- **归属：** 它属于**每一个独立的对象**。
- **特性：** 你创建多少个 `Dog` 对象，内存中就有多少份 `weightInPounds`。
- **作用：** 用来描述每个对象**各自的状态**。
### 2. `static` (静态) 变量

也称为“类变量”。

- **归属：** 它属于 `Dog` **这个类**。
- **特性：** 无论你创建了多少个 `Dog` 对象（即使一个也不创建），内存中**永远只有一份** `static` 变量。
- **作用：** 用来存储这个类的**“全局”状态或共享信息**。
### 总结

- **`non-static` 变量 (实例变量)：**
    
    - **用途：** 每个对象**独有**的数据（如：体重、名字、颜色）。
        
    - **访问：** `对象名.变量名` (e.g., `dog1.weightInPounds`)。
        
- **`static` 变量 (类变量)：**
    
    - **用途：** 所有对象**共享**的数据（如：总数 `count`、物种名称 `species = "Canis lupus"`）。
        
    - **访问：** `类名.变量名` (e.g., `Dog.count`)。
